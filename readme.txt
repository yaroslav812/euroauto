 Основные файлы проекта:

index.php - скрипт создания таблицы category с данными
db_data.sql - SQL dump базы для Postgress
db_config.php - настройка параметров БД
categories.html - страница тестирования сервиса


 Для настройки стенда на локальном сервере требуется:
 
1. Отредактировать файл db_config.php для подключения к Postgres.

2. Запустить index.php

3. Выбрать действие загрузки данных в базу
   Произойдет попытка удаления и создания таблицы "category" и заполнение ее данными. 

4. Перейти по ссылке на страницу с деревом категорий


 Средства использованные в разработке:

- PHP 5.x
- PostgreSQL 9.4
- JQuery
- Bootstrap


 Протестировано в:

- Google Chrome
- Firefox
- Internet Explorer
